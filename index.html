Leaflet Button

Add a simple button to a Leaflet map.

L.control.button({
    title: "Refresh map", 
    icon: 'fa fa-street-view ', 
    func: function(){
        map.setView(
            [49.264012,-123.095931], 
            16, 
            {animate: true}
        );
        marker.openPopup()
    }
}).addTo(self.map);
